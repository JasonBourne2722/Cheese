# Cheese Pepe ETH Site

Open `index.html` in a browser to view the page.

## Add your images

Put your assets in `site/assets/`:

- `character-main.png` (hero character image)
- `chronicle-1.png` (newspaper image, unchanged)
- `chronicle-2.png` (newspaper image, unchanged)
- `chronicle-3.png` (newspaper image, unchanged)

You can also add more character art and reuse it in `index.html`.
